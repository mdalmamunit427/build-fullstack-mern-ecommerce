Course start date

A great project grabs your attention from the start. This course begins with the fundamentals and guides you through every step of building a full-stack MERN e-commerce website, from initial setup to final deployment. It’s everything you need to create a robust online store, except the coffee (we haven’t figured out how to code that in yet).

Week One

Frontend

We start with the empty folder. You'll design and create a user-friendly shopping experience, including things like product categories and a shopping cart. You’ll also make user register and login funtionality to authentication.

Technology: React Js, Tailwind Cs, Reduxt, Redux RTK Query, Remixicon, and other.

Week Two

Backend

In this 2nd week’s, we’ll look at the backend that you need to sprinkle into your backend skills. You'll work with Node.js and Express.js to handle server-side logic, manage user authentication, and process payments securely with Stripe. Use MongoDB and Mongoose to store and retrieve data, including user information, orders, product details and reviews.

Technology: Node, Expressjs, MongoDB, Mongoose, JWT, Dotenv, Bcrypt, Body-parser, etc.

Week Three

User and Admin Dashboard

The User Dashboard lets customers manage their orders, update their profiles, and review purchased products with ease. Meanwhile, the Admin Dashboard provides a powerful control panel for managing products, tracking inventory, processing orders, and analyzing site performance through visual reports.

Technology: Chart Js, React Charts and Date Formater, Dashboard Graphical Interface

Week Four

Deployment

Deploy your first e-commerce website with ease by learning how to publish your project on Vercel. This course covers the steps to deploy your MERN stack application, ensuring it's live and accessible to users. You’ll understand how to configure your deployment settings, manage updates, and maintain your site for optimal performance and reliability.

Technology: GitHub and Vercel

Week Five

Description

A good story has a good place at its heart. But how do you bring your settings to life? We’ll analyse some of the key techniques to help you write dynamic description.

Session Topics: Senses, Movement, Colour, Small Details, People and the Past.
